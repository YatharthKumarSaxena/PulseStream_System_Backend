📊 DATA FLOW
═══════════════════════════════════════════════════════════════════════════

Frontend (React-like)
     ↑↓
Socket.IO (WebSocket)
     ↑↓
Backend (Node.js)
     ↑↓
Redis (Data Storage)

User selects patient → Frontend emits "selectPatient"
                   ↓
          Backend fetches from Redis
                   ↓
          Backend calculates statistics
                   ↓
          Backend sends "healthData" event
                   ↓
          Frontend receives and updates UI
                   ↓
          Charts and metrics display in real-time

🔌 SOCKET.IO INTEGRATION
═══════════════════════════════════════════════════════════════════════════

Frontend → Backend (Events Emitted):

  socket.emit('selectPatient', 'patient_123');
  socket.emit('changeWindow', 30);  // 15, 30, or 60

Backend → Frontend (Events Received):

  socket.on('healthData', (data) => {
    // data.current: { bpm, spo2, temp, humidity }
    // data.stats: { avg: {...}, min: {...}, max: {...} }
  });
